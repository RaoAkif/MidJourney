## Getting Started

To get a local copy of the project, follow these steps: 
Clone the repository with
 ```js
 git clone https://github.com/tkazi/CreativeHat
 ```

 ```js
 cd CreativeHat
 ```

Install the packages by running the commands
```js
npm i
```
